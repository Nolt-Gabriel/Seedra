##  Sobre o Projeto
 
O **SEEDRA** é uma aplicação desenvolvida para facilitar o gerenciamento e consulta de espécies vegetais. Seja para hortas domésticas, viveiros, ou coleções botânicas, o sistema centraliza todas as informações essenciais de cada planta ou semente em um único lugar, de forma organizada e acessível.


## Funcionalidades principais
 
-  **Catálogo completo** de espécies de plantas e sementes
-  **Controle de estoque/quantidade** por espécie
-  **Fichas detalhadas** com informações de cuidado (rega, luz, solo, temperatura)
-  **Busca e filtragem** por nome, tipo, ou características
-  **Cadastro e edição** de novas espécies
-  **Categorização** por tipo (hortaliça, fruto, ornamental, medicinal, etc.)

## Instalação e Execução

**Pré-requisitos**
- `Os requisitos estão presentes no arquivo requirements.txt`

**Passo a Passo**

```bash
# 1 - Clone o repositório
git clone  https://gitlab.com/pdsi3/seedra.git
cd seedra

# 2 - Configure as variáveis de ambiente
python -m venv .venv
source  .venv/Scripts/activate

# 3 - Instale as dependências
# (pip install -r requirements.txt)

# 3 - Execute a aplicação
flask run --debug
```
